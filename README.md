# Ilya Runner

Browser game with Ilya as the main character.

## Play Online

Open the game here:  
[https://shulenok.github.io/david/](https://shulenok.github.io/david/)

## Controls

- `Space` or right mouse button: jump (double jump)
- `F`: boost jump
- `Shift` or `E`: nitro
- `S`: stop
- `R`: restart
